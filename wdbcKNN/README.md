## Wisconsin Breast Cancer Classifier
O notebook foi criado no Google Colab, e pode ser aberto diretamente por lá.

O dataset está disponíveis nos formatos CSV e Excel. O script está configurado para ler o arquivo no formato Excel.

Para reproduzir o script:
* clique no arquivo wdbcKNN.jpynb - o script será aberto
* clique no botão **Open in Colab**
* execute o script

O script já está configurado para buscar os dados no github.

Se quiser obter uma cópia dos dados basta baixar o arquivo wdbc.xlsx.
