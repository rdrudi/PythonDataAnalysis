## Haberman KNN
O notebook foi criado no Google Colab, e pode ser aberto diretamente por lá.  
O dataset está no formato original - CSV, e é acessado diretamente pelo script.  
Para reproduzir o script:
* clique no arquivo HabermanKNN.jpynb - o script será aberto
* clique no botão **Open in Colab**
* execute o script

O script já está configurado para buscar os dados no github.  
Se quiser obter uma cópia basta baixar o arquivo haberman.data
